# Conexión RedJIV · Plan

Perfil bajo por diseño · documento vivo · español e inglés

Este repositorio contiene todo el plan de acción de la red ante los terremotos del 24 de junio de 2026: la página de presentación y, debajo, cada documento en detalle, en dos idiomas. La página `index.html` es la puerta de entrada; los documentos están en `docs/`.

Low profile by design · living document · Spanish and English

This repository holds the network's full earthquake action plan: the landing page and, underneath it, every document in detail, in two languages. The page `index.html` is the front door; the documents live in `docs/`.

## Cómo usar / How to use

1. Lee la página o el documento 1. Read the page or document 1.
2. Sigue el plan de acción, documento 3, para crear la organización, asegurarla e invitar a la gente. Follow the action plan, document 3, to create the organization, secure it and invite people.
3. Súmate por el intake que enlaza la página. Join through the intake linked on the page.

## Publicar la página / Publish the page

Para servirla como web: Settings, Pages, Deploy from branch, main, root. Si prefieres mantener el repositorio privado, comparte el PDF impreso de la página, que evita cualquier bloqueo de sitios.

To serve it as a site: Settings, Pages, Deploy from branch, main, root. If you prefer to keep the repository private, share the printed PDF of the page, which avoids any site blocking.

## Nota / Note

Antes de abrir el intake y mover datos de miembros o de pacientes, conviene una revisión de protección de datos, sobre todo por los miembros en la UE. No es asesoría legal.

Before opening the intake and moving member or patient data, a data-protection review is advisable, especially given members in the EU. This is not legal advice.
